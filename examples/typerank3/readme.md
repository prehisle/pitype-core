# 我要做一个打字练习软件 
##　v0.1版本目标，路径如下
1. 仅使用最基础的html+javascript+css来实现
2. 加载预定义的10段预定义的英文文本中的一段显示
3. 初始光标停在第一个字符左侧
4. 在用户输入第一个字符后，如何输入正确，将延续一个字符设置为绿色，光标跳到第二个字符左侧，以此类推，直到输入完成
5. 练习过程中实时显示cpm、正确率、本次练习时间
6. 完成练习后，弹出统计报告，按回车可以重新开始练习
7. 隐藏输入框，仅用于捕获键盘输入
