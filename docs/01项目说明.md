# 01 项目说明

## 背景

- `examples/typerank3` 是一个纯 JavaScript 的打字练习器，集成了文本拆分、实时统计、主题/语言切换等完整体验。
- `examples/svelte-typerank3` 通过 Svelte + TypeScript 复刻 ts-typerank3 的所有交互，展示与 DOM Hook 的结合方式。
- 为了让更多团队在此基础上扩展自定义 UI 或终端应用，我们把其中的核心能力抽象成 npm 包 `pitype-core` 并统一放在 monorepo 中维护。

## 目标

1. **核心无 UI 引擎**：提供 TypingSession、StatsTracker、文本管线与 DOM 适配器等模块，让任意框架都能复用。
2. **示例 & 文档**：保留 JS/TS/Vue3 示例，作为官方接入样板，同时提供接入指南、测试指南、架构规划等文档。
3. **工程化**：引入 TypeScript、Vitest、Playwright、ESLint/Prettier、GitHub Actions、semantic-release，保证质量和自动发布。

## 当前状态

- `packages/pitype-core` 已产出 TypingSession、StatsTracker、tokenizer、语言/主题 API、DOM 适配器等模块，并配置了 `npm run task:menu` 等脚本。
- `examples/typerank3`、`ts-typerank3`、`vue3-typerank3`、`react-typerank3`、`next-typerank3`、`svelte-typerank3` 均基于核心包运行，Playwright 基线测试覆盖中文输入、错误回退、主题/语言切换、移动端滚动等场景。
- 文档体系包含：README（总览）、QUICK_REFERENCE（命令速查）、DEVELOPMENT（开发/CI）、docs/03（规划）、docs/04（接入指南）、docs/05（测试指南）。
